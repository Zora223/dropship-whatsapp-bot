# 🤖 Dropship Perú - WhatsApp Bot

Backend Node.js con Baileys para envío automático de mensajes de WhatsApp.

## 🚀 Setup Local

```bash
# 1. Instalar dependencias
npm install

# 2. Copiar variables de entorno
cp .env.example .env
# (Editar .env con tu API_KEY)

# 3. Iniciar en desarrollo
npm run dev

# 4. Escanear QR desde WhatsApp
